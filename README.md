# Quiz Project

A simple console-based quiz application built in Java. Test your knowledge with 5 multiple-choice questions and get your score at the end!

## Features

- Interactive quiz with multiple-choice questions
- Score calculation and display
- Case-sensitive answer checking

## How to Run

1. Ensure you have Java installed on your system.
2. Compile the Java files:
   ```
   javac src/*.java
   ```
3. Run the application:
   ```
   java -cp src Main
   ```

## Project Structure

- `src/Main.java`: Entry point of the application
- `src/QuestionService.java`: Handles quiz logic, question display, and scoring
- `src/Questions.java`: Represents a quiz question with options and answer

## Technologies Used

- Java
